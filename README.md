# Albuquerque

## What is this?
Albuquerque is my personal hobbyist graphics engine. The goal is to self-learn and experiment with computer science concepts (especially with graphics) without being derivative on traditional and popular online tutorials, as such [LearnOpenGL](https://learnopengl.com/). It is hence an attempt to outgrow bad habits I had developed as a beginner.

It uses [Deccer's Template](https://github.com/deccer/CMake-Glfw-OpenGL-Template) as a starting point.
